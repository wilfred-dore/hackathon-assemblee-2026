# data/fixtures/ — jeux de démo

Dépose ici 1–2 lois/textes pré-téléchargés pour la démo (JSON, texte, ou
export Canutes). Ces fichiers sont **git-ignorés** (sauf ce README) : ils
peuvent être volumineux et éviter d'appeler le réseau pendant la démo.

Idées :
- une loi récente (ex. version consolidée + version initiale) pour illustrer
  « La loi après la loi » ;
- un article de code pour tester le retrieval Canutes hors-ligne.
